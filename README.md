# Practice — 楽器練習トラッカー

複数の楽器の練習時間・内容を記録し、GitHub風のヒートマップや累計グラフで進捗を可視化するWebアプリです。

---

## 実行方法

**必要なもの:** Node.js 18以上

```bash
# 依存パッケージをインストール（初回のみ）
npm install

# 開発サーバーを起動
npm run dev
```

起動後、ブラウザで `http://localhost:5173` を開いてください。

```bash
# 本番ビルド
npm run build

# ビルド済みファイルをプレビュー
npm run preview
```

## 機能一覧

### ホーム画面
- 当月の楽器別 月間目標プログレスバー
- サマリーカード（今月の合計 / 累計 / 連続練習日数🔥）
- GitHub風 芝生ヒートマップ（過去6ヶ月）
- 最近の練習記録リスト

### カレンダー画面
- 月間カレンダー表示（日本の祝日対応）
- 各日セルに楽器タグ・時間を表示
- 日付クリックでその日の記録詳細を展開

### グラフ画面
- **累計**: 楽器別の練習時間成長曲線
- **月別**: 楽器ごとのグループ棒グラフ
- **週別**: 週単位の棒グラフ
- 表示期間: 1ヶ月 / 3ヶ月 / 6ヶ月 / 1年 / 全期間
- 楽器フィルター切り替え

### 楽器別詳細画面
- ホームの楽器カードをクリックで遷移
- その楽器だけのヒートマップ・累計グラフ
- 練習記録の一覧・削除

### 目標設定画面
- 楽器ごとの月間目標時間（時間/月）を設定
- 目標変更は翌月から適用（過去データに影響しない）

### 設定画面
- 楽器の追加 / 削除 / 表示切り替え
- カラーカスタマイズ
- ダーク / ライト テーマ切り替え
- ヒートマップ濃淡の閾値設定
- データのエクスポート（JSON）/ インポート（JSON）
- 全データ削除

### 練習記録の入力

**タイマーモード**（「練習開始」ボタン）
1. タイマーをスタート
2. 一時停止・再開が可能
3. 終了後に楽器・メモ・曲名・動画を入力して保存

**手入力モード**（タイマーモーダル内のタブ）
- 日付・楽器・分数を直接入力して保存

## データ

データはすべてブラウザの `localStorage` に保存されます。ブラウザをまたいだ同期はありません。

- 設定画面の **JSONエクスポート** でバックアップを取得できます
- **JSONインポート** で別のブラウザ・PCに移行できます

### デフォルト楽器

| 楽器 | 略称 | カラー |
|---|---|---|
| ギター | Gt. | #E8A32E（amber）|
| ベース | Ba. | #1D9E75（teal）|
| ドラム | Drs. | #7F77DD（purple）|

初回起動時はベースとドラムが表示状態、ギターは非表示です。
